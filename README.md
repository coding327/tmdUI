# tmd-ui 组件库

### 组件文档

[文档链接](https://coding327.gitee.io/tmdui-docs)

### 快速开始

#### 1. 安装组件库

```bash
npm i tmd-ui
```

#### 2. 引用组件库
```js
// 完整引入
import TMDUI from 'tmd-ui'
import 'tmd-ui/dist/css/index.css'
Vue.use(TMDUI)

// 按需引入
import { Demo } from 'tmd-ui'
import 'tmd-ui/dist/css/demo.css'
Vue.use(Demo)
```
